# Ink Panel

> **Ink Inc. presents** — AI Creation, Human Care. The Future Drawn Together.

IRIAM配信者向け  
**パネル開け企画作成ツール**

画像1枚からパネルを作成し、ギフト設定・進行管理・X投稿までを  
ブラウザだけで行うことができます。

インストール不要・サーバー不要・APIキー不要。  
`index.html` をブラウザで開くだけで動作します。

---

## Features

- 一枚絵からパネル作成
- 空白自動パネル化
- ギフト自動割り当て
- 非ギフト設定
- 全部達成パネル
- プレビュー機能
- X投稿用画像生成
- Undo / Redo

すべて **ブラウザのみで動作します。**

サーバー処理・AI・外部サービスは一切使用していません。

---

## 使い方

```
1. index.html をダウンロード
2. ブラウザで開く
3. 画像をアップロードしてパネルを作成
```
